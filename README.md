# Usb микрофонная решетка

# Задача

Реализовать USB микрофонную решетку с асинхронным типом синхронизации данных

# Реализация

* MCU - CH32V307
* Dev board - CH32V307V-EVT-R1
* Частота дискретизации звука - 48 кГц

# Тонкости текущей реализации 

* Важно затактировать все блоки от одного источника сигнала
  * Для таргетной частоты дискритизации звука нужен кварт частотой 24.576 Мгц
  * На текущей devboard'е HSE кварц расположен на самой плате. Из-за этого требует поставить сам кварц на devboard'у.
    * Y1 кварц, что нужно заменить
  * Альтернатива. [Статья](https://www.cnblogs.com/wchmcu/p/17880995.html) про конфигурация частоты в 24.576 МГц из кварца 8 МГц
* для записи 4 микрофонов используем два I2S, один из которых настроен как master, а другой как slave с подсоединёнными тактирующими сигналами от master'а
* на отладочной плате CH32V307VCT6-R1 high speed девайс находится справа от ETH(не там где стоит LDO)
